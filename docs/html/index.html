Development environment¶

To start developing on the project you should:

  1. Clone the repo if you haven’t already : git clone @github.com:triglex/papernews.git

  2. Create and activate a new virtual environment: ?

  3. Install development requirements pip3 install -r requirements-dev.txt

  4. Run tests to check if everything is working make ci
